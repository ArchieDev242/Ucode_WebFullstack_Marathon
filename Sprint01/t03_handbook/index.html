HTML

Introduction to HTML

HTML is the standard markup language for creating web pages.

<!DOCTYPE html>

Example:


<html>
    <head>
        <title>Title</title>
    </head>

    <body>
        <h1>Hello, World!</h1>
    </body>
</html>

HTML Elements

HTML documents are composed of elements represented by tags.

<p>This is a paragraph</p>

Example:


<div>
    <h2>Heading</h2>
    <p>Paragraph</p>
</div>

HTML Attributes

Attributes provide additional information about elements.

<img src="image.jpg" alt="Image">

Example:

<a href="https://www.example.com">Link</a>

HTML Forms

Forms allow users to input data for server submission.

<form action="/submit" method="post">

Example:

<input type="text" name="username">

CSS

Introduction to CSS

CSS is used for describing the presentation of HTML documents.

body { background-color: lightblue; }

Example:


body 
{ 
    font-family: Arial, sans-serif; 
}

CSS Selectors

Selectors are patterns used to target elements for styling.

p { color: blue; }

Example:


#header 
{ 
    font-size: 24px; 
}

CSS Box Model

The box model describes element spacing, including padding, border, and margin.

.box { padding: 10px; }

Example:


.box 
{ 
    border: 1px solid black; 
}

CSS Flexbox

Flexbox is a layout model that arranges elements within a container.

.container { display: flex; }

Example:


.item 
{ 
    flex-grow: 1; 
}

Protocols

HTTP/HTTPS

HTTP and HTTPS are protocols for data transfer on the web. Methods and response codes are described in detail.

DNS

DNS translates domain names to IP addresses and includes various record types.

IP

IP defines the rules for data transmission over the internet and methods for determining an IP address.
